# 参考答案与解析

## 第 1 题

x² − 5x + 6 = 0，因式分解 (x−2)(x−3) = 0，故 x₁ = 2, x₂ = 3。

## 第 2 题

2x² + 3x − 2 = 0，因式分解 (2x−1)(x+2) = 0，故 x₁ = 1/2, x₂ = −2。
