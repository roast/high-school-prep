# 一元二次方程（巩固难度）

## 第 1 题

解方程 $x^2 - 5x + 6 = 0$。

\vspace{6\baselineskip}

## 第 2 题

解方程 $2x^2 + 3x - 2 = 0$。

\vspace{6\baselineskip}

\newpage

# 参考答案与解析

## 第 1 题

$x^2 - 5x + 6 = 0$，因式分解 $(x-2)(x-3) = 0$，故 $x_1 = 2,\ x_2 = 3$。

## 第 2 题

$2x^2 + 3x - 2 = 0$，因式分解 $(2x-1)(x+2) = 0$，故 $x_1 = \frac{1}{2},\ x_2 = -2$。
