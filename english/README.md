# 英语 — 高一预习

## 教材
人教版高中英语必修第一册（2019）— 高一上学期

每单元按 PEP 标准七模块：Listening and Speaking / Reading and Thinking / Discovering Useful Structures / Listening and Talking / Reading for Writing / Assessing Your Progress / Video Time。

- Welcome Unit（初高衔接）
  - Listening and Speaking
  - Reading and Thinking
  - Discovering Useful Structures（基本句型与成分）
  - Reading for Writing
- Unit 1 Teenage Life
  - Reading and Thinking（青少年生活对比）
  - Discovering Useful Structures（状语从句与建议表达）
  - Reading for Writing（建议信写作）
- Unit 2 Travelling Around
  - Reading and Thinking（旅行计划与目的地）
  - Discovering Useful Structures（将来时态：will / be going to / be doing）
  - Reading for Writing（旅行邮件/行程写作）
- Unit 3 Sports and Fitness
  - Reading and Thinking（体育精神与健康）
  - Discovering Useful Structures（反身代词与情态动词）
  - Reading for Writing（个人经历叙事）
- Unit 4 Natural Disasters
  - Reading and Thinking（自然灾害与救援）
  - Discovering Useful Structures（定语从句：that / which / who 引导）
  - Reading for Writing（事件摘要写作）
- Unit 5 Languages Around the World
  - Reading and Thinking（世界语言与汉字传承）
  - Discovering Useful Structures（定语从句：where / when / 介词+which/whom）
  - Reading for Writing（语言学习经验分享）

## 科内专属规则
- 可切换全英文对话鼓励开口
- 阅读材料生词率≤10%，超纲词必注释
- 语法点优先用例句对比，不堆术语

## 文件命名约定
- 知识点：`knowledge/<单元>-<知识点>/README.md`（welcome 单元用 `welcome-<知识点>`）
- 演示：`knowledge/u<单元>-<知识点>/demo.html`
- Python 可视化：`knowledge/u<单元>-<知识点>/viz/<主题>.py`
- 题目：`questions/YYYY-MM-DD_<知识点>_<难度>_<序号>.md`

## 进度文件
- `progress.md`：知识点状态表 + 当日总结
- `mistakes.md`：错题索引
