## 科内专属规则
- 物质的量为计算核心，建立"宏观-微观-符号"三重表征
- 化学方程式须配平且标注状态（s/l/g/aq）
- 用类比化解抽象概念（摩尔↔打、阿伏伽德罗常数↔一打数量）
