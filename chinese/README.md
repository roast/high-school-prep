# 语文 — 高一预习

## 教材
人教版高中语文必修上册
- 第一单元：青春的价值（诗词）
- 第二单元：劳动光荣（新闻/通讯）
- 第三单元：生命的沉思（古诗文）
- 第六单元：学习之道（议论性古文）
- 第七单元：自然情怀（散文）

## 科内专属规则
- 古诗文先讲背景故事再读原文
- 议论文训练重逻辑框架，检查论点-论据匹配
- 文言文逐字落实重点实词虚词，忌整句翻译代替字词
