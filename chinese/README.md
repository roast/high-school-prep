# 语文 — 高一预习

## 教材
人教版高中语文必修上册（2019）— 高一上学期

- 第一单元 青春的价值（现代诗词）
  - 1. 沁园春·长沙
  - 2. 红烛 / 立在地球边上放号 / 峨日朵雪峰之侧 / 致云雀
  - 单元写作：青春的价值
- 第二单元 劳动光荣（新闻与通讯）
  - 4. 喜看稻菽千重浪——记首届国家最高科学技术奖获得者袁隆平
  - 5. 心有一团火，温暖众人心（记张秉贵）
  - 6. "探界者"钟扬
  - 单元写作：写人要凸显个性
- 第三单元 生命的沉思（古典诗词）
  - 7. 短歌行 / 归园田居（其一）
  - 8. 梦游天姥吟留别 / 登高 / 琵琶行（并序） / 念奴娇·赤壁怀古 / 永遇乐·京口北固亭怀古 / 声声慢
  - 单元写作：文学短评
- 第四单元 乡土中国（整本书阅读）
  - 把握核心概念 / 调研提纲 / 主题研讨
- 第六单元 学习之道（议论性古文）
  - 10. 劝学（《荀子》）
  - 11. 师说（韩愈）
  - 12. 赤壁赋（苏轼）
  - 13. 谏太宗十思疏（魏征）
  - 单元写作：议论要有针对性
- 第七单元 自然情怀（散文）
  - 14. 故都的秋（郁达夫）
  - 15. 荷塘月色（朱自清）
  - 16. 我与地坛（节选，史铁生）
  - 单元写作：散文的景与情

## 科内专属规则
- 古诗文先讲背景故事再读原文
- 议论文训练重逻辑框架，检查论点-论据匹配
- 文言文逐字落实重点实词虚词，忌整句翻译代替字词
